서버란
서버는 정보를 저장하고 관리하는 강력한 컴퓨터로, 다른 컴퓨터나 장치에 정보를 제공하는 역할을 한다. 우리가 인터넷을 사용할 때 필요한 데이터나 서비스를 주고받는 중심 역할을 한다. 프론트엔드란
프론트엔드는 웹사이트나 앱에서 사용자가 보는 화면과 상호작용하는 부분을 만드는 작업이이다. HTML, CSS, JavaScript 등을 사용해 디자인과 기능을 구현한다. 백엔드란
백엔드는 웹사이트나 앱에서 사용자가 보지 못하는 부분으로, 서버와 데이터베이스를 관리하고, 데이터를 처리하며, 프론트엔드와 연결하는 역할을 한다. 주로 서버 언어(예: Python, Java, Node.js 등)로 개발한다.
